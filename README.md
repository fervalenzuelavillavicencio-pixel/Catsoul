Mi Proyecto: Tienda Online Catsoul.
¿Qué es?
Es una tienda online de poleras y polerones bordados en general de mascotas. 

¿Qué tiene?
-4 páginas web conectadas entre sí

-Catálogo de productos y precios

-Carrito de compras que recuerda lo que agregas.

-Diseño con colores negro y dorado.

Funciona en celular y computadora

Tecnologías que usé:
-HTML → para la estructura

-CSS → para los colores y diseño

-JavaScript → para que el carrito funcione
